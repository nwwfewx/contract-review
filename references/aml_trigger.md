# AML Trigger

## AML / Anti-Bribery Checks
Only trigger review if the contract already includes AML/反洗钱 or anti-bribery/反商业贿赂 clauses.
If present but vague, flag and suggest clarifying language.
Do NOT require adding these clauses if the contract is silent, unless the user explicitly requests it.
See `references/aml_anti_bribery_checklist.md` for the minimal checklist and wording guidance.
