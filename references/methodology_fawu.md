# Fawu Methodology Summary

## Orientation
1. Contract review serves the next scenario, not only current signature.
2. Prioritize real transaction control over formal completeness.

## Priority Stack
1. Target clauses
2. Process clauses
3. Protection clauses
4. Auxiliary clauses

## Role Strategy
1. Strong side: lock certainty and execution efficiency.
2. Weak side: build fallback rights and evidence leverage.
3. Balanced deal: preserve flexibility while controlling downside.

## Control Outside Contract
1. Credit control
2. Process and evidence control
3. Performance monitoring control
4. Internal governance control

## Five Scenario Elements
1. Time
2. Place
3. Actor
4. Plot/transaction path
5. Background constraints
