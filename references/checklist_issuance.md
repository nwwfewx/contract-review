# Issuance-Side Checklist 发行端审核清单

> 适用范围：保险资管产品发行端（组合类/债权投资计划/股权投资计划/ABS）
> 
> 与投资端差异：发行端关注管理人视角，投资端关注委托人/投资人视角
>
> 叠加规则：若合同为托管协议/托管合同，需在本清单基础上叠加 `references/checklist_custody_agreement_2026.md`。

---

## 一、产品设立合规

### 1.1 备案要求
- [ ] 产品备案主体资格（是否具备管理人资质）
- [ ] 备案材料完整性（产品合同、风险揭示书、投资者承诺书）
- [ ] 备案时限（设立后 5 个工作日内）

### 1.2 产品投资范围
- [ ] 投资范围是否符合产品类型限制
- [ ] 是否存在禁止投资标的（如地方政府隐性债务）
- [ ] 集中度限制是否满足（单一资产/单一交易对手）

### 1.3 产品结构
- [ ] 产品分级安排（如有）是否合规
- [ ] 嵌套层级是否超限
- [ ] 资金穿透到底层资产的路径是否清晰

### 1.4 资金用途与监测
- [ ] 资金用途与合同约定一致，禁止归集/挪用/超范围补流
- [ ] 资金用途穿透与监测机制充分（账户安排/用途凭证/报告频率）

### 1.5 主体性质与治理约束
- [ ] 关键主体（融资主体/增信主体/项目公司/管理人）股权结构与实控人穿透清晰
- [ ] 上市公司参与或控制的，核查信息披露/关联交易/资金占用与违规担保限制
- [ ] 金融/类金融机构参与的，核查监管要求、资产隔离与关联交易限制
- [ ] 地方国企/平台主体涉及的，核查地方财政/国资委特别规定与审批程序
- [ ] 担保/增信/融资决策的股东会或董事会授权充分且有效

---

## 二、管理人权责边界

### 2.1 管理职责
- [ ] 投资决策权归属是否明确
- [ ] 投资经理/投委会权限边界
- [ ] 风控措施的执行责任

### 2.2 免责条款
- [ ] 免责事由是否过宽（管理人责任限缩）
- [ ] 是否存在"一切损失由投资人承担"等不合理条款
- [ ] 不可抗力定义是否合理

### 2.3 费用与报酬
- [ ] 管理费/托管费/销售服务费计提方式
- [ ] 业绩报酬计提基准及分配顺序
- [ ] 费用承担主体是否明确

---

## 三、投资者适当性

### 3.1 合格投资者
- [ ] 投资者准入门槛（金融资产/投资经验）
- [ ] 穿透核查要求（多层嵌套时）
- [ ] 投资者人数限制（≤200 人）

### 3.2 风险揭示
- [ ] 风险揭示书内容完整性
- [ ] 投资者签署确认流程
- [ ] 特殊风险（流动性/集中度/信用）是否充分揭示

### 3.3 适当性匹配
- [ ] 产品风险等级与投资者风险承受能力匹配
- [ ] 冷静期安排（如适用）

---

## 四、信息披露合规

### 4.1 定期披露
- [ ] 季报/年报披露时限
- [ ] 净值披露频率
- [ ] 披露渠道（官网/中保登）

### 4.2 临时披露
- [ ] 重大事项披露触发条件
- [ ] 披露时限要求
- [ ] 关联交易披露

### 4.3 信息送达
- [ ] 送达方式（邮件/短信/系统）
- [ ] 视为送达条款是否合理

---

## 五、运营与终止

### 5.1 申购/赎回
- [ ] 开放期安排
- [ ] 巨额赎回处理机制
- [ ] 暂停申购/赎回条件

### 5.2 产品终止
- [ ] 提前终止条件
- [ ] 清算程序
- [ ] 剩余财产分配顺序

### 5.3 管理人更换
- [ ] 更换条件
- [ ] 交接程序
- [ ] 投资者权益保护

---

## 六、关联审查与法人独立性（独立版基线）

> 以下检查点为本仓内置基线；若有外部模板库，可作为可选叠加而非前置依赖。

- [ ] 法人独立性检查（SPV/通道/项目公司是否具备独立运营与独立财务）
- [ ] 融资主体、增信主体、项目公司之间股权与控制关系可穿透
- [ ] 上市公司/金融机构参与时的资金占用、违规担保与资产混同风险
- [ ] 关联交易合规与定价公允性
- [ ] 资金监管/托管安排可执行且可监测
- [ ] 增信措施有效性（担保/抵质押/承诺函等）
