# Evidence Completeness Gate

用途：校验证据链是否足以支撑交付意见。  
执行方式：按条检查；任何 P0/P1 结论缺证据默认 FAIL。

## Gate Items
- [ ] E1 每条风险意见均有条款位置（章节/条款号或可定位锚点）
- [ ] E2 每条风险意见均有原文摘录（非二手转述）
- [ ] E3 每条风险意见均有风险后果（触发机制清楚）
- [ ] E4 每条风险意见均有修改建议（可执行，不是抽象提醒）
- [ ] E5 关键结论含依据状态（明确依据或 `【需人工复核】`）
- [ ] E6 版本冲突点有优先级判定（签署版/补充协议/草稿）
- [ ] E7 不可比比较已降级为并行观察（无赢家/输家话术）
- [ ] E8 资管场景已覆盖法人独立性/关联交易/资产隔离检查

## PASS Criteria
1. P0/P1 条目 100% 满足 E1-E5。  
2. 其余条目满足率不低于 90%。  
3. 未满足项必须写入“未闭环清单”。
