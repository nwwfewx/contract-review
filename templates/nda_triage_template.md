# NDA Quick Triage Template

Use only when the document is an NDA/保密协议, unless the user requests full review.

## Triage Output
- 协议类型：单向 / 双向
- 风险等级：GREEN / YELLOW / RED
- 关键缺口（最多5条）：
- 是否进入完整审查：是 / 否

## GREEN Criteria (examples)
- 标准豁免条款齐全（公开信息/先前知悉/独立开发/第三方合法取得/法定披露）
- 期限合理（1–3年）且存续期合理（2–5年）
- 无非竞争、排他、禁招、残留条款等附加义务
- 争议解决地合理

## YELLOW Criteria (examples)
- 缺少1项标准豁免
- 期限偏长但可谈
- 保密信息定义略宽但可通过红线修订

## RED Criteria (examples)
- 缺少多个关键豁免
- 包含非竞争/排他/禁招等不应出现在NDA的条款
- 过度宽泛的保密信息定义或永久保密义务
