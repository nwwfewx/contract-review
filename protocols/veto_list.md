# Veto List 一票否决清单

> 触发任一条目即阻断审查流程，升级为 P0 处理

## Veto Items 一票否决事项

| 编号 | 类型 | 风险描述 | 识别特征 | 处理结果 |
|------|------|----------|----------|----------|
| V01 | 隐债/平台 | 涉及地方政府隐性债务或融资平台 | 借款人/担保人为城投公司、财政兜底条款 | **建议不予投资** |
| V02 | 资金用途不可监测 | 无法穿透资金最终流向 | 无专户、无监管银行、归集账户共用 | **建议不予投资** |
| V03 | 法人独立性重大瑕疵 | SPV无实质运营、资产混同 | 无独立办公、员工为借调、现金流混同 | **建议不予投资** |
| V04 | 核心授权缺失 | 担保/处分无有效授权决议 | 缺董事会/股东会决议、超权限担保 | **暂停审查→补齐后继续** |
| V05 | 强制执行效力存疑 | 公证/仲裁条款无效或不完整 | 管辖约定矛盾、仲裁机构不明确 | **降级为高风险标注** |
| V06 | 期限错配严重 | 资金来源与投资期限严重不匹配 | 短融长投超过1年 | **标注合规风险** |
| V07 | 明股实债穿透 | 穿透后实质为债权但以股权形式包装 | 固定回报+保底安排+回购承诺 | **需合规定性后继续** |

## Detection Triggers 识别触发词

### V01 隐债/平台
- 城市建设投资公司、城投、经开投、开发区投资
- 财政承诺函、财政兜底、纳入预算
- 政府购买服务、棚改项目、土储贷款

### V02 资金用途不可监测
- 资金池、归集使用、统借统还
- 无专户监管、自主划付
- 用途变更无需审批

### V03 法人独立性重大瑕疵
- 项目公司无员工、借用母公司场地
- 财务人员为派遣、共用银行账户
- 印章由股东保管

### V04 核心授权缺失
- 超越权限、未经审议
- 授权文件缺失、决议过期
- 法定代表人无权签署

### V05 强制执行效力存疑
- 仲裁机构约定模糊（如"北京仲裁"）
- 管辖法院与仲裁条款并存
- 公证债权文书要素不全

## Output Format 否决报告格式

```markdown
## ⛔ 一票否决触发

**触发类型**：V01 隐债/平台
**触发位置**：借款人基本信息 / 担保安排
**识别依据**：
- 借款人名称含"城市建设投资"
- 担保函提及"纳入市级财政预算"

**处理建议**：建议不予投资

**如需继续审查**：
请提供以下材料排除隐债风险后重新提交：
1. 借款人企业性质证明（市场化转型文件）
2. 项目不涉及隐性债务的声明函
3. 地方财政局出具的无隐债确认函
```

## Integration 与其他规则的联动

- 触发 V01-V03 → 直接终止审查，仅输出否决报告
- 触发 V04 → 暂停审查，等待材料补充
- 触发 V05-V07 → 继续审查，在风险清单中置顶标注
