# Investment Advisor Agreement Checklist 投资顾问协议审核清单

> 适用场景：保险资管产品管理人聘请外部投资顾问（含私募证券类管理人、证券公司、基金管理人等）提供投资建议服务。
>
> 使用方式：在 `general` 或 `asset_mgmt:issuance` 路由基础上叠加本清单；如同时存在产品合同、募集说明书、风险审查表，以三者交叉核对。

## 一、资质与准入

- [ ] 投资顾问主体身份、牌照/登记、业务范围与本产品投资标的匹配。
- [ ] 如投资顾问为私募基金管理人，已核查其登记状态、自律处分情况及适用的协会规则，协议中未通过“投资顾问”安排规避私募基金监管或自律要求。
- [ ] 协议要求投资顾问在合作期间持续保持资质有效，并在资质中止、注销、限制展业、会员资格变化时及时报告。
- [ ] 投资顾问、其实际控制人及核心人员与管理人、融资方、担保方、底层资产服务方的关联关系已披露；如存在关联，已设置单独审批和隔离安排。
- [ ] 协议要求投资顾问披露近三年监管处罚、失信记录、重大诉讼/仲裁及其后续进展。
- [ ] 如管理人与该投资顾问属于首次合作，已核查是否触发并完成相应的监管报告或内部前置审批要求。

## 二、管理人主动管理与权责边界

- [ ] 明确“投资顾问仅提供建议，管理人独立审查并作出最终投资决策”。
- [ ] 明确管理人有权拒绝、暂缓、调整、终止采纳任何投资建议，不因未采纳而构成违约。
- [ ] 明确管理人保留独立尽调权、风险监测权、止损/减仓/暂停投资权，不得通过投资顾问形式变相转委托或让渡主动管理职责。
- [ ] 明确投资顾问不得通过提供投资建议或交易方案设计的方式规避法律法规、监管规则、自律规则、产品合同或投资限制要求。
- [ ] 协议未出现“必须采纳”“默认执行”“投资顾问代替管理人判断”等通道化表述。
- [ ] 投资顾问不得直接联系投资者，不得开展路演、推介、投诉处理或收益承诺。

## 三、投资建议机制

- [ ] 投资建议的内容、格式、有效期、发送方式、接收对象、确认方式明确。
- [ ] 投资建议至少覆盖：标的、方向、数量/金额、价格/估值、时间、交易对手/交易场所、适用产品、授权人员信息。
- [ ] 投资建议需包含合规性声明：符合监管规定、产品合同、募集说明书、投资范围、投资比例、关联交易和公平交易要求。
- [ ] 投资建议范围、投资限制和关键监督口径宜以附件、监督事项表或其他书面清单固化，并与产品合同、募集说明书保持一致。
- [ ] 如存在多名授权人员，协议应约定“不一致建议暂停生效，待投资顾问重新确认”；不得将相互矛盾的建议一并视为有效。
- [ ] 原件、扫描件、传真件不一致时的处理规则清楚；优先规则不得削弱管理人的真实性核验权。
- [ ] 建议函签署标准足以识别真实性，优先要求“授权人员签字/签章 + 机构预留印鉴”；仅凭个人签字即生效的，应视为高风险。
- [ ] 对当日交易建议设置合理截点与补正机制，避免压缩管理人独立审查时间。

## 四、费用与激励

- [ ] 固定投资顾问费率、浮动报酬、承担主体、支付路径与产品合同/募集说明书一致。
- [ ] 若管理人提取业绩报酬，需核查投资顾问是否构成重复提取或变相由产品直接向管理人与投顾双重支付。
- [ ] 投资顾问费若由产品资产承担，应在募集说明书中充分披露；若由管理费支付，应与内部审批口径一致。
- [ ] 投资顾问激励机制不得诱导高换手、激进投资、利益输送或规避监管。

## 五、信息隔离与利益冲突

- [ ] 协议要求投资顾问建立信息隔离墙，不得将本产品信息用于其他业务。
- [ ] 协议明确禁止抢跑、老鼠仓、内幕交易、操纵市场、利益输送。
- [ ] 协议明确禁止推荐关联方资产；如确需推荐，必须先披露关联关系并经管理人独立审批。
- [ ] 若投资顾问同时服务多个产品/账户，协议应要求公平分配投资机会，不得优先照顾关联或收费更高的账户。

## 六、合规报告与应急处置

- [ ] 协议要求投资顾问按季度报告合规状况、监管检查情况和重大负面事项。
- [ ] 协议要求投资顾问在监管处罚、牌照/登记变化、核心人员变更、重大诉讼/仲裁、重大舆情等事项发生后在约定时限内报告。
- [ ] 协议明确顾问失资质、核心人员失联/离岗、合规事件爆发时，管理人有立即接管决策、暂停采纳建议、解除协议的权利。
- [ ] 协议约定终止后的交接义务：资料返还/销毁、未了事项移交、历史建议和沟通记录交付。

## 七、责任、终止与争议解决

- [ ] 管理人单方解除权覆盖：资质不符、重大违规、重大失实陈述、利益冲突未披露、服务能力明显不足、拒不配合尽调/整改等情形。
- [ ] 投资顾问对违法违规建议、虚假资料、未披露关联关系、保密义务违反、抢跑/利益输送等行为承担赔偿责任。
- [ ] 协议不存在投资顾问的绝对免责、过低责任标准或与控制能力不匹配的责任上限。
- [ ] 争议解决地、送达条款、证据保全安排对管理人可操作。

## 八、产品文件与内部留痕联动

> 以下项目可以不全部写入协议正文，但至少应进入补证清单或内部留痕清单。

- [ ] 募集说明书已披露投资顾问身份、角色边界、费用安排、关联关系。
- [ ] 定期报告模板已预留投资顾问履职情况、费用支付、重大变更披露口径。
- [ ] 管理人已建立投资决策会议/投决留痕机制，能够证明独立审查过程。
- [ ] 已建立投资建议采纳率统计与预警机制；采纳率持续过高应触发合规复核。
- [ ] 已保留“不采纳建议”的书面说明和独立尽调复核记录。

## 红线提示

- [ ] 不得出现“100%采纳投资顾问建议”的制度或事实状态。
- [ ] 不得完全依赖投资顾问尽调、估值或交易判断替代管理人独立判断。
- [ ] 不得通过投资顾问安排规避主动管理责任、关联交易审查或投资范围限制。
